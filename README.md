##  Project Overview

The objective of this project was to:

- Recreate a historic newspaper front page
- Maintain traditional 1970s print-style layout
- Use only HTML and inline CSS
- Preserve classic newspaper typography
- Stack content exactly as seen in original scans
- Simulate aged paper background and vintage formatting

This project was created as an academic assignment to demonstrate layout structuring and historical design recreation using basic web technologies.
